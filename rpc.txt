

Open(path string) (int, error)
    allocate file handle and return id
Write(handle int, buf []byte) error
    write buffer to file buf
    maybe flush
Close(handle int)
    flush file buffer
        if no vlog:
            for each vlog in bucket:
                try acquiring lock; break
            else:
                make new vlog, acquire lock
        for chunk in buffer:
            write to vlog under lock

MakeVlog(params) vlogid
Write(vlog, txnid, buf []byte) (offset, error)
Read(vlog, offset, length) ([]byte, error)
Commit(txnid int) error

MakePlog(params) plogid
Write(plog, txnid, lockid int, buf []byte) (offset, error)
Read(plog, offset, length) ([]byte, error)
Commit(txnid int) error

tables:

chunks: list of 120b hashes + 32b lengths. ~19kB/GB overhead
vaddr: 32b vlogid + 32b offset
paddr: 32b plogid + 32b offset

plog(plogid, disk, length)
sectormetahash(paddr, hashes)
